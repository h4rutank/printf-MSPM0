# printf-MSPM0
This is a library to implement kind of printf() on MSPM0

# objectives
To implement kind of printf() over URAT and conduct printf-debugging on serial-monitor.<br>
This is not redirect to printf() in stdio.h.<br>
Just Simple and Fast.

# Dev Environment 
* [LP-EM-MSPM0L2228](https://www.ti.com/tool/LP-MSPM0L2228)
* Windwos 11
* CCS theia
 
# Usage
* Create CCS project
* Add library files to project directory, "uart_print.h" and "uart_print.c"
* Include the library in main.c
* Call "printf_()" in main.c
